# SillyTavern TTS Sequencer

这是一个面向 SillyTavern 的轻量浏览器扩展。它会在检测到 SillyTavern 页面后，在右侧显示一个可拖动的悬浮气泡。点击气泡时，扩展会自动播放最新对话楼层里的所有 TTS/语音按钮或原生音频。

## 安装方式

### Chrome / Edge

1. 打开扩展管理页：
   - Chrome：`chrome://extensions/`
   - Edge：`edge://extensions/`
2. 打开“开发者模式”。
3. 点击“加载已解压的扩展”。
4. 选择这个文件夹：

   `C:\Users\ASUS\Documents\酒馆拓展\side-button-audio-sequencer`

## 使用方式

- 打开 SillyTavern 页面。
- 页面检测成功后，右侧会出现一个绿色悬浮气泡。
- 点击气泡：播放最新对话楼层里的所有 TTS 语音。
- 播放过程中再次点击气泡：暂停当前顺序播放。
- 按住气泡拖动：调整位置，位置会保存在当前浏览器里。

## 说明

- 扩展虽然安装在浏览器里，但只有检测到 SillyTavern 聊天界面时才会显示和工作。
- 播放目标会限定在最新的 SillyTavern 消息楼层中，不再扫描整个网页。
- 扩展会同时识别标准 `<audio>` 音频元素，以及常见 TTS/语音/播放按钮。
- 每段语音之间会加入接近正常对话的停顿：短句停顿较短，长句后停顿略长，并带有轻微随机变化。
- 对部分自定义语音按钮，扩展会通过浏览器调试接口发送真实点击。Edge 顶部可能会显示“正在调试此浏览器”的提示，这是正常现象。
- 如果某个 TTS 插件使用了很特殊的按钮结构，可能还需要针对它的按钮 class 或标题做进一步适配。
