# Floating Audio Sequencer

这是一个轻量浏览器扩展：在网页右侧生成一个小型圆形悬浮按钮。点击按钮时，自动按页面从上到下的顺序播放当前浏览器显示区域里的音频。

## 安装方式

### Chrome / Edge

1. 打开扩展管理页：
   - Chrome：`chrome://extensions/`
   - Edge：`edge://extensions/`
2. 打开“开发者模式”。
3. 点击“加载已解压的扩展”。
4. 选择这个文件夹：

   `C:\Users\ASUS\Documents\酒馆拓展\side-button-audio-sequencer`

## 使用方式

- 打开包含音频控件的网页。
- 点击页面右侧的小圆形悬浮按钮，扩展会顺序播放当前浏览器显示区域里的音频。
- 播放过程中再次点击悬浮按钮，会暂停/停止当前顺序播放。

## 说明

- 浏览器只允许在用户操作后自动播放声音，所以第一次播放需要由你点击悬浮按钮触发。
- 当前版本会识别网页里的标准 `<audio>` 音频元素，也会尝试识别带有 `5"`、`16"`、`5秒` 这类时长标记的自定义语音气泡。
- 点击悬浮按钮后会先显示当前显示区域的扫描结果，例如“标准音频 0，语音气泡 5”。
- 页面没显示出来的音频不会播放；需要先滚动到那一段，再点击悬浮按钮。
- 每段音频之间会保留很短的自然停顿，模拟角色对话的节奏，同时避免等待过长。
- 悬浮按钮会广播给当前标签页里的所有 iframe 一起尝试扫描。
- 扩展会扫描 iframe 和 shadow DOM，所以嵌在聊天框架里的语音按钮也会尝试识别。
- 对自定义语音气泡，扩展会通过浏览器调试接口发送真实点击。Edge 顶部可能会显示“正在调试此浏览器”的提示，这是正常现象。
- 如果某个网站的播放器完全阻止脚本触发点击，可能还需要针对那个网页做专门适配。
