# SillyTavern Latest Message TTS Sequencer

一个面向 SillyTavern 的原生第三方扩展，用来一键播放最新对话楼层里的所有 TTS 语音。

它适合使用角色语音、TTS 气泡、语音胶囊或嵌入式 `<audio>` 的酒馆用户：当一层回复里包含多段语音时，只需要点击页面上的悬浮气泡，扩展就会按楼层内顺序逐段播放，并自动加入接近真人对话的停顿。

## 核心能力

- **一键播放最新楼层**：点击悬浮气泡后，自动扫描最新对话楼层中的所有语音。
- **识别多种 TTS 形态**：支持标准 `<audio>`、常见播放按钮、TTS 按钮、语音按钮，以及 `4"`、`4”`、`4″`、`4秒` 这类时长胶囊。
- **自然对话间隔**：短语音后停顿较短，长语音后停顿略长，并加入轻微随机变化。
- **不打断阅读位置**：播放时尽量不滚动聊天视野。
- **最近语音楼层兜底**：如果最新楼层没有语音，会自动回退到最近一个带 TTS 的楼层。
- **可拖动气泡**：悬浮气泡可以拖动，位置会保存在浏览器本地。
- **手机端适配**：支持手机浏览器远程访问 SillyTavern 时使用，按钮会避开移动端输入栏和安全区。
- **GPT-SoVITS 气泡适配**：可直接播放 `.voice-bubble` 语音胶囊，等待生成完成后再顺序播放，减少手机端点击后无声的问题。
- **原生酒馆扩展**：仓库根目录就是 SillyTavern 可安装扩展，不需要用户手动加载浏览器扩展。

## 通过 GitHub URL 安装

1. 打开 SillyTavern。
2. 进入 **Extensions** / **扩展** 面板。
3. 点击 **Install Extension** / **安装扩展**。
4. 粘贴本仓库 GitHub 地址：

   `https://github.com/Fantome4174/sillytavern-tts-sequencer`

5. 安装完成后刷新 SillyTavern 页面。

这个地址可以直接复制到 SillyTavern 的扩展安装器中。

## 使用方式

安装并刷新 SillyTavern 后，聊天界面会出现一个绿色悬浮气泡。

- 点击气泡：播放最新楼层的所有 TTS 语音。
- 播放中再次点击：暂停当前顺序播放。
- 按住气泡拖动：调整位置。

播放开始时会显示识别到的语音数量，例如“开始播放最新楼层：3 段 TTS”。

### 手机端使用

手机通过远程地址打开电脑版 SillyTavern 后，扩展会自动使用移动端按钮位置和触摸事件。若按钮位置不合适，可以拖动到顺手的位置；手机端和电脑端的位置会分开保存。

如果手机端看不到悬浮气泡，或能看到按钮但点了不播放，请先在 SillyTavern 的扩展管理里更新本扩展到 `1.3.2` 或更新到最新版本，然后刷新手机页面。新版会在页面主体加载后立即显示按钮，并对 GPT-SoVITS 的 `.voice-bubble` 语音胶囊使用专用播放路径。

## 适配说明

扩展会优先在最新楼层中查找语音。识别目标包括：

- 页面里的 `<audio>` 元素。
- 带有 TTS、voice、audio、speaker、play、volume 等语义的按钮。
- 中文标题或标签里包含“朗读 / 播放 / 语音 / 音频 / 声音 / 喇叭 / 听”的按钮。
- 仅显示时长的小型语音胶囊，例如 `1"`、`3”`、`4″`、`5秒`。
- GPT-SoVITS 生成的 `.voice-bubble` 气泡，包含 `data-audio-url`、`data-status`、`data-key` 的语音胶囊。

如果某个 TTS 插件使用了非常特殊的 DOM 结构，可能需要继续补充 class 或属性适配。

## 调试

在浏览器控制台运行：

```js
window.STTtsSequencer.scanLatest()
```

它会返回当前识别到的楼层和语音列表，方便判断是否识别到了目标 TTS 气泡。

也可以直接运行：

```js
window.STTtsSequencer.playLatest()
```

这会触发一次最新楼层语音播放。

## 文件结构

- `manifest.json`：SillyTavern 原生扩展清单。
- `index.js`：扩展主逻辑。
- `style.css`：悬浮气泡和提示样式。
- `browser-extension/`：旧版 Chrome/Edge 浏览器扩展，作为兼容版本保留。

## 兼容版本

仓库根目录是推荐使用的 SillyTavern 原生扩展。

`browser-extension/` 目录中保留了旧版 Chrome/Edge 浏览器扩展。如果你确实需要浏览器扩展形式，可以手动加载该目录；普通用户不需要使用它。
